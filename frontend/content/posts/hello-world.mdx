Welcome to my blog. I built this site with Next.js, Tailwind CSS, and MDX — blog posts are just markdown files that live alongside the code.

## Why a blog?

Writing is a great way to solidify understanding. If I can explain something clearly, I probably understand it. If I can't, I need to dig deeper.

I plan to write about things I'm working on, tools I find useful, and ideas worth exploring.

## Stack

This site is built with:

- **Next.js 14** — App Router, server components
- **Tailwind CSS** — utility-first styling
- **shadcn/ui** — accessible component primitives
- **MDX** — markdown with embedded components
- **Flask** — backend API (minimal for now)

```python
# The backend is deliberately minimal
from flask import Flask

app = Flask(__name__)

@app.route("/health")
def health():
    return {"status": "healthy"}
```
